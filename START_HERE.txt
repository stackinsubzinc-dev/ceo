🛠️ TROUBLESHOOTING:

Server not running?
  cd backend && python server.py

Can't see endpoints?
  Check MongoDB is running
  Check .env file has all API keys

Gumroad not working?
  Verify GUMROAD_API_TOKEN in .env
  Test: curl http://localhost:8000/gumroad/status

TikTok not posting?
  Verify TIKTOK_CREATOR_TOKEN in .env
  Test: curl http://localhost:8000/api/social/tiktok/analytics/channel/summary
