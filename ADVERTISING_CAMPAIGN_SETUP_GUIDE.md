# Advertising Campaign Automation System - Setup & Implementation Guide

## Overview

The Advertising Campaign Automation System enables **one-click multi-platform ad campaign management** with intelligent product discovery, real-time performance tracking, and ROI optimization.

### Key Features

✅ **Product Ranking Engine** - Automatically identify top-performing products
✅ **Multi-Platform Campaign Manager** - Launch ads across 7 platforms simultaneously  
✅ **Revenue Attribution** - Track which campaigns generate which sales
✅ **ROI Optimization** - Real-time performance metrics and recommendations
✅ **Time-Series Analytics** - Monitor campaign profitability over time

---

## System Architecture

### 1. Product Ranking Engine (`product_ranking_engine.py`)

**Purpose:** Identify the best products for advertising investment

**Ranking Metrics:**
- REVENUE - Total sales revenue
- SALES_COUNT - Number of units sold  
- CONVERSION_RATE - Percentage of visitors who bought
- MARGIN - Profit per sale
- TRENDING - Week-over-week growth velocity
- ENGAGEMENT - Social signals and reviews
- REVIEWS - Customer ratings
- VELOCITY - How fast products are selling

**Key Methods:**

```python
# Get top 10 products by revenue (last 30 days)
top_products = await ranker.get_top_products(
    limit=10,
    metrics=["revenue", "sales_count"],
    time_period="30d"
)

# Get fastest growing products
trending = await ranker.get_trending_products(limit=5)

# Get most profitable products
high_margin = await ranker.get_high_margin_products(limit=10)

# Get product readiness for advertising
health = await ranker.get_product_health(product_id)
```

**Output Score: 0-100**
- 80+: STRONGLY RECOMMENDED - High sales, good margin, social proof
- 60+: RECOMMENDED - Good product, ready for ads
- 40+: CONDITIONAL - Needs more reviews first
- <40: NOT RECOMMENDED - Build social proof first

### 2. Multi-Platform Ad Campaign Manager (`multi_platform_ad_manager.py`)

**Purpose:** Create and manage ad campaigns across all platforms

**Supported Platforms:**

1. **Google Ads** - Search, Display, Shopping ads
2. **Facebook Ads** - Feed, Stories, Reels ads
3. **TikTok Ads** - Native video ads with high engagement
4. **LinkedIn Ads** - B2B targeting, professional audience
5. **Pinterest Ads** - Visual discovery, high intent buyers
6. **Amazon Ads** - Product ads with built-in purchasing
7. **YouTube Ads** - Pre-roll, bumper, non-skippable formats

**Campaign Creation:**

```python
result = await manager.create_campaign(
    product_id="prod_123",
    platforms=["google_ads", "facebook_ads", "tiktok_ads"],
    budget=500.00,           # Total budget for all platforms
    daily_budget=50.00,      # Per platform daily budget
    duration_days=30,
    target_audience={
        "age_min": 18,
        "age_max": 65,
        "interests": ["shopping", "lifestyle"],
        "countries": ["US", "CA"]
    }
)

# Result:
{
    "campaign_id": "camp_abc123xyz",
    "status": "created",
    "product_id": "prod_123",
    "platforms_created": {
        "google_ads": {"status": "created", "platform_id": "..."},
        "facebook_ads": {"status": "created", "platform_id": "..."},
        "tiktok_ads": {"status": "created", "platform_id": "..."}
    }
}
```

**Campaign Management:**

```python
# Get real-time performance
perf = await manager.get_campaign_performance(campaign_id)
# Returns: impressions, clicks, conversions, spend, CTR, CPC, ROI

# Get optimization recommendations
optimize = await manager.optimize_campaign(campaign_id)
# Returns: AI recommendations for creative, audience, budget

# Pause campaign
result = await manager.pause_campaign(campaign_id)
```

### 3. Revenue Attribution Engine (`revenue_attribution_engine.py`)

**Purpose:** Track revenue generated by campaigns and calculate ROI

**Attribution Models:**

- **LAST_TOUCH** - Campaign that led directly to purchase (default)
- **FIRST_TOUCH** - Initial campaign that introduced user to product
- **LINEAR** - Equal credit to all touchpoints
- **TIME_DECAY** - Recent interactions weighted more heavily
- **POSITION_BASED** - 40% first, 40% last, 20% middle touches

**Recording Sales:**

```python
# When a customer purchases (triggered by payment webhook)
result = await attribution.record_sale(
    product_id="prod_123",
    amount=29.99,
    campaign_id="camp_abc123xyz",  # Links to ad campaign
    platform="google_ads",
    user_id="user_xyz"
)
```

**ROI Calculation:**

```python
# Get comprehensive ROI for campaign
roi = await attribution.get_campaign_roi(
    campaign_id="camp_abc123xyz",
    attribution_model="last_touch"
)

# Returns:
{
    "campaign_id": "camp_abc123xyz",
    "total_spend": 500.00,
    "attributed_revenue": 2500.00,
    "profit": 2000.00,
    "roi_percent": 400.0,       # $1 spent = $4 revenue
    "payback_period_days": 7.5
}
```

**Platform Breakdown:**

```python
# See which platform performed best
breakdown = await attribution.get_platform_attribution(campaign_id)

# Returns:
{
    "by_platform": {
        "google_ads": {
            "spend": 150,
            "revenue": 900,
            "roi_percent": 500
        },
        "facebook_ads": {
            "spend": 200,
            "revenue": 700,
            "roi_percent": 250
        },
        "tiktok_ads": {
            "spend": 150,
            "revenue": 900,
            "roi_percent": 500
        }
    },
    "best_performer": "tiktok_ads"
}
```

**Time Series Tracking:**

```python
# Monitor ROI progression daily
time_series = await attribution.get_time_series_roi(campaign_id, interval="daily")

# Shows cumulative spend, revenue, and ROI each day
# Helps identify when campaigns become profitable
```

## Usage Examples

### Example 1: Find Top Products and Run Campaigns

```python
# 1. Find best products
ranker = await get_product_ranking_engine()
top_products = await ranker.get_top_products(limit=5)

for product in top_products:
    print(f"Product: {product['title']}")
    print(f"Revenue: ${product['metrics']['total_revenue']}")
    print(f"Ranking Score: {product['ranking_score']}/100")
    
    # 2. Create campaign for each
    manager = await get_campaign_manager()
    result = await manager.create_campaign(
        product_id=product['product_id'],
        platforms=['tiktok_ads', 'google_ads'],  # High performers
        budget=200,
        daily_budget=20,
        duration_days=14
    )
    
    print(f"Campaign created: {result['campaign_id']}")
```

### Example 2: Monitor Campaign Performance

```python
# Check performance daily
performance = await manager.get_campaign_performance(campaign_id)

metrics = performance['aggregated_metrics']
print(f"Spend: ${metrics['total_spend']}")
print(f"Impressions: {metrics['total_impressions']}")
print(f"Clicks: {metrics['total_clicks']}")
print(f"Conversions: {metrics['total_conversions']}")
print(f"ROI: {metrics['estimated_roi_percent']}%")

# Get optimization tips
recommendations = await manager.optimize_campaign(campaign_id)
for rec in recommendations['recommendations']:
    print(f"[{rec['type']}] {rec['action']}")
    print(f"Expected impact: {rec['expected_impact']}")
```

### Example 3: Revenue Dashboard

```python
# Get complete revenue overview
dashboard = await attribution.get_revenue_dashboard()

print(f"Total Campaigns: {dashboard['total_campaigns']}")
print(f"Total Spend: ${dashboard['financial_summary']['total_spend']}")
print(f"Total Revenue: ${dashboard['financial_summary']['total_revenue']}")
```

## Troubleshooting

### Issue: No conversions recorded

**Solution:** Verify payment system is recording campaign_id and linking to sales endpoint

### Issue: Low CTR on all platforms

**Solution:** Run optimize_campaign endpoint to get creative refresh recommendations

### Issue: Campaign not created

**Solution:** 
1. Check platform API credentials are configured
2. Verify product exists in database
3. Check daily_budget is reasonable ($5+ typical minimum)

### Issue: ROI calculations don't match expectations

**Solution:** 
1. Confirm sales are being recorded to sale endpoint
2. Check attribution_model (default: last_touch)
3. Verify campaign_id is being stored with payments
