🚀 GETTING STARTED (5 STEPS)
═══════════════════════════════════════════════════════════════════════

1. Get TikTok API CREDENTIALS
   → https://developers.tiktok.com
   → Create App
   → Get Client ID, Client Secret, API Key, Access Token

2. CONFIGURE ENVIRONMENT
   $ cp .env.tiktok.example .env
   $ # Edit .env with your credentials

3. START BACKEND
   $ cd backend
   $ python server.py

4. CREATE & POST PRODUCT
   POST /api/products/{id}/post-tiktok-series
   {
     "title": "My Product",
     "description": "...",
     "price": 97,
     "series_count": 5
   }

5. CHECK RESULTS
   GET /api/social/tiktok/analytics/channel/summary
