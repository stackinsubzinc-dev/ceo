## 🎯 STEP-BY-STEP (Choose ONE store to start)

### **Option A: Use Gumroad (FASTEST - Already Set Up!)**
✅ Your Gumroad is already configured in the system
1. Go to: https://gumroad.com (your dashboard)
2. Check: Do you have ANY products uploaded?
   - If YES → Proceed to Step 2
   - If NO → Create a test product now (takes 2 min)
3. In FiiLTHY, go to: **Products** page → Click **"🔄 Refresh"**
4. Your Gumroad products should appear! ✓

### **Option B: Connect Shopify (RECOMMENDED - 10 minutes)**
1. Do you have a Shopify store?
   - If NO → Go to https://www.shopify.com and start a free trial
   - If YES → Continue below

2. In Shopify Admin:
   - Go to: Settings → Apps and Integrations → Develop Apps
   - Click: "Create an app"
   - Name it: `FiiLTHY Sync`
   - Permissions needed:
     - ✓ read_products
     - ✓ read_orders  
     - ✓ read_inventory
   - Install the app
   - Copy your **Admin Access Token**

3. In FiiLTHY:
   - Go to: **Vault** (https://frontend-one-ashen-16.vercel.app/vault)
   - Find: **Shopify** card
   - Click: **Connect**
   - Paste: Your **Admin Access Token**
   - Paste: Your store URL (e.g., `mystore.myshopify.com`)
   - Click: **Test Connection**

4. Go to **Products** page and refresh → Your Shopify products appear! ✓

### **Option C: Connect Etsy (EASY - 5 minutes)**
1. Do you have an Etsy shop?
   - If NO → Go to https://www.etsy.com/seller and create one
   - If YES → Continue below

2. Get your credentials:
   - Go to: https://www.etsy.com/developers
   - Create an App
   - Get your **API Key** and **API Secret**

3. In FiiLTHY:
   - Go to: **Vault**
   - Find: **Etsy** card
   - Click: **Connect**
   - Paste credentials
   - Click: **Test Connection**

4. Your Etsy shop products appear in FiiLTHY! ✓

## 📱 NOW FOR SOCIAL MEDIA (Display Ad Posts)

### **Instagram (PRIORITY #1)**
Once you connect Instagram, FiiLTHY will:
- ✅ Post automatically to your feed
- ✅ Show your posts in the Social Media page
- ✅ Track engagement real-time

Steps (10 minutes):
1. Is your Instagram a Business Account?
   - Go to: instagram.com → Settings → Account Type
   - Switch to: Business (if not already)
   - Select category: Creator or Business

2. Create a Meta app:
   - Go to: https://developers.facebook.com
   - Create app → Type: Business
   - Add product: Instagram Graph API
   - Get your **Access Token**:
     - Advanced Tab → Access Token Tool
     - Generate token with permissions:
       ✓ pages_manage_posts
       ✓ pages_read_engagement
       ✓ instagram_basic
       ✓ instagram_graph_manage_pages

3. Get your Instagram Business Account ID:
   - Instagram app → Settings → About Account
   - Find your Business Account ID

4. In FiiLTHY:
   - Go to: **Vault**
   - Find: **Instagram** card
   - Click: **Connect**
   - Paste: **Access Token**
   - Paste: **Business Account ID**
   - Test Connection

5. Go to **Social Media** page → See your Instagram feed + scheduled posts! ✓

### **TikTok (PRIORITY #2)**
Steps (10 minutes):
1. Do you have a TikTok creator account?
   - Go to: https://www.tiktok.com/creators/
   - Set up creator account

2. Create API credentials:
   - Go to: https://developer.tiktok.com
   - Create app
   - Get: **Client Key** and **Client Secret**
   - Get: Your **TikTok User ID** (from profile in iOS/Android app)

3. In FiiLTHY:
   - Go to: **Vault**
   - Find: **TikTok** card
   - Click: **Connect**
   - Paste credentials
   - Test Connection

4. Go to **Social Media** page → See TikTok posted videos! ✓
